\section{Заключение}
В настоящем методическом пособии рассмотрен вывод формул метода конечных элементов на примере задачи об одномерном деформировании линейно-упругого стержня под действием постоянной во времени нагрузки. Рассмотрено две физические постановки задачи:
\begin{itemize}
    \item заданы условия Дирихле на обоих концах стержня;
    \item на левом конце задано условие Дирихле, а на правом конце задано условие Неймана.
\end{itemize}
Получены матрицы жёсткости и вектора силы как для отдельного конечного элемента, так и глобальные. В явном виде выписаны их значения для случая, когда параметры стержня и внешняя нагрузка не зависят от координаты вдоль его оси. Показана процедура перехода от линейных базисных функций к квадратичным для повышения точности численного решения.

Для продолжения изучения МКЭ может быть использована следующая литература:
\begin{itemize}
    \item The Finite Element Method: Linear Static and Dynamic Finite Element Analysis, T.J.R. Hughes, Dover Publications, 2000.
    \item The Finite Element Method: Its Basis and Fundamentals, O.C. Zienkiewicz, R.L. Taylor and J.Z. Zhu, Butterworth-Heinemann, 2005.
    \item A First Course in Finite Elements, J. Fish and T. Belytschko, Wiley, 2007.
\end{itemize}
