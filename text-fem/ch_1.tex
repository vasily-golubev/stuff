\section*{Введение}
\addcontentsline{toc}{section}{Введение}

Множество физических явлений может быть описано с использованием системы уравнений в частных производных. К сожалению, поиск аналитического решения для произвольной геометрии расчётной области зачастую невозможен. Для получения численного решения может быть использован метод конечных элементов (МКЭ). Анализ напряжённо-деформированного состояния, задача о переносе тепла, задача о движении флюида или электромагнетизм - вот типичные области применения данного метода.

Основной идеей МКЭ является разбиение расчётной области на подобласти, называемые конечными элементами, которые соединены между собою узлами. Этот процесс называется построением расчётной сетки. В дальнейшем, в каждом из элементов решение аппроксимируется некоторым набором базисных функций с неизвестными весами. МКЭ позволяет записать систему линейных уравнений на эти неизвестные. Решая её находятся веса, по которым в дальнейшем однозначно восстанавливается решение исходной задачи. При этом для повышения точности решения требуется вводить всё более мелкое разбиение области на подобласти.

МКЭ изначально был разработан в 1930-x -- 1950-х годах. К его основателям относят Хренникова А.П., R. Courant, R.W. Clough, E. Wilson, R.L. Taylor, O.C. Zienkiewicz. Интересно, что задача о сходимости метода была решена лишь десять лет спустя (для линейного случая и гладкого решения). В 1965 году при поддержке NASA была разработана программа NASTRAN (Dick MacNeal) с большим функционалом по решению 2D и 3D задач, а в 1969 - программа ANSYS (John Swanson), ABAQUS в 1978. Ещё представителем МКЭ решателя является LS-DYNA, разработанная в 1989 году (Livermore National Laboratory, John Hallquist). Сегодня существует множество библиотек, предоставляющих функционал МКЭ (например, https://www.dealii.org).
