{\bfseries Введение в метод конечных элементов. Одномерная статическая упругость.}: учебно-методическое пособие
/ сост. В.И.~Голубев. -- М.:~МФТИ,~2019. -- 24~с.

\vspace{10mm}

Работа посвящена методу конечных элементов, активно использующемуся для решения статических и динамических задач упругости.
На примере задачи об одномерном нагружении упругого стержня с переменным сечением изложена процедура построения определяющей системы линейных алгебраических уравнений МКЭ.
Рассмотрены различные граничные условия, случай постоянных параметров задачи вдоль оси стержня.
Все выкладки приводятся для линейных и квадратичных базисных функций.
Предназначается для студентов, изучающих информатику и вычислительную математику, в виде практики по реализации вычислительных алгоритмов.
