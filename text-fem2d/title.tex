{\bfseries Введение в метод конечных элементов. Двумерная статическая теплопроводность.}: учебно-методическое пособие
/ сост. В.И.~Голубев, А.В.~Борисова. -- М.:~МФТИ,~2020. -- 26~с.

\vspace{10mm}

Работа посвящена методу конечных элементов, активно использующемуся для решения статических и динамических задач упругости.
На примере задачи о двумерном статическом распределении тепла изложена процедура построения определяющей системы линейных алгебраических уравнений МКЭ.
Рассмотрены граничные условия первого рода, приведены аналитические решения для изотропного тензора теплопроводности.
Все выкладки приводятся для простейших квадратных элементов линейных базисных функций.
Предназначается для студентов, изучающих вычислительную математику, в виде практики по реализации вычислительных алгоритмов.
