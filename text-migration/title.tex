{\bfseries Построение миграционных изображений в упругих средах}: учебно-методическое пособие
/ сост. В.И.~Голубев. -- М.:~МФТИ,~2018. -- 32~с.

\vspace{10mm}

Работа посвящена вычислительным алгоритмам построения миграционных изображений гетерогенных геологических сред.
Рассматривается формула Кирхгофа и метод Борна, позволяющий линеаризовать связь между наблюдаемыми значениями и параметрами модели.
Выкладки проводятся как для акустического приближения, так и для полноволнового упругого описания среды.
Предназначается для студентов, изучающих информатику и вычислительную математику, в виде практики по математическому моделированию.
