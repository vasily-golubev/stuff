\section{Сигналы для некогерентной локации}

Импульсный локатор (\emph{радар или сонар}) передает зондирующий сигнал с комплексной огибающей $x(t)$. Огибающая принятого
отражения
\begin{equation}\label{app:0}
x^T(t) = x(t-\tau_T)e^{j2\pi \, f_Tt}e^{j2\pi \, \theta}
\end{equation}
отличается задержкой $\tau_T$, допплеровским сдвигом по частоте $f_T$ и неконтролируемой начальной фазой.

\vspace{0.2cm}
\textbf{$\blacktriangle$ Для начала}

\begin{itemize}
\item{
Проанализировать спектры и корреляции импульсов прямоугольник ($\textbf{r}$), манчестер ($\textbf{m}$).}

\item{
Наблюдая спектр и корреляцию импульса ($\textbf{s}$), оценить ширины частотного и временного окон ДПФ. (\emph{Для этого
потребуется временно увеличить $X$-диапазоны}).}

\item{
Для импульса ($\textbf{c}$) изучить спектр $FFT(v(p)*\exp(6*j*pi*t))$. Объяснить наблюдаемое явление. }
\end{itemize}

\textbf{$\blacktriangle$ Исследовательские задания}

\begin {enumerate}
\item{Выяснить, при каком значении параметра $A$ первый нуль спектра импульса ($\textbf{e}$) совпадает с ($\textbf{s}$).}
\item{Выяснить, при каком значении параметра $B$ первый нуль спектра ($\textbf{cp}$) совпадает с первым нулем спектра ($\textbf{s}$).}
\item{Выяснить, при какой постоянной времени $R_1C_1$ происходит подавление вдвое импульса ($\textbf{m}$).}
\end {enumerate}

Гипотеза проверяется по отклику на выходе фильтра, согласованного с ожидаемым сигналом $x^H(t)$, то есть по значению скалярного
произведения принятой реализации $x^T(t)$~на~$x^H(t)$:
\[\left<x^T,x^H\right> = e^{j2\pi \, \theta} \int x(t-\tau_T)x^*(t-\tau_H) e^{j2\pi \,(f_T-f_H)t} dt = \]
\[ = e^{j2\pi \, \theta} \int x(t)x^*(t-(\tau_H-\tau_T)) e^{-j2\pi \,(f_H-f_T)t} dt.\]

Скалярное произведение оказывается функцией от текущих ошибок по задержке $\tau = \tau_H-\tau_T$ и частоте $f =
f_H-f_T$.


\subsection {Линейная частотная модуляция,\\ чирп-преобразование}

Исторически первый метод сжатия зондирующего импульса основывается на чирп-преобразовании, вызывающем линейную
модуляцию частоты.

Чирп с параметром $D$~--- это гармонический сигнал с линейно изменяющейся частотой
\[c_D(t) = e^{-j 2\pi  \frac{D}{2}t^2} = e^{-j \pi Dt^2}.\]
Мгновенная частота этого сигнала, определенная как производная от фазы
\[f(t) = \frac{d \frac{D}{2}t^2}{dt} = Dt,\]
линейно нарастает со временем. Параметр $D$ имеет смысл скорости изменения частоты~---~герц в секунду.

\begin{center}
  \includegraphics[width=1.0\textwidth]{pictures/radar_1.jpg}
  \caption {К обсуждению чирп-преобразования} \label{radar:1}
\end{center}
\end{figure}

Рис.~\ref{radar:1}(b) иллюстрирует природу деформации тела неопределенности при чирпировании. Преобразование координат
$(\tau',f') = (\tau,f+D\tau)$ переводит каждую горизонтальную прямую $f=F$ в наклонную~---~$f' = F+D\tau$ (см.~рис.~\ref{YaRM}). Вертикальный же
отрезок, удаленный от начала координат на $\tau$, сдвигается по вертикали вдоль оси частот вверх-вниз на $D\tau$. В результате
получается что-то вроде преобразования шарнирно-скрепленного прямоугольника в параллелограмм.
