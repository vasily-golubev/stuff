{\bfseries Построение сеточно-характеристических схем для линейного одномерного уравнения переноса}: учебно-методическое пособие
/ сост. В.И.~Голубев. -- М.:~МФТИ,~2013. -- 22~с.

\vspace{10mm}

Работа посвящена построению численных сеточных-характеристических методов для решения одномерного линейного уравнения переноса, которое возникает в задачах
переноса взвеси частиц в жидкости, газовой динамике и динамических задачах упругого тела.
Предназначается для студентов, изучающих информатику, в виде практики по реализации вычислительных алгоритмов на языке Си.
